## ***Project : Landing Page*** 



#### **Title : The Earth**  



##### **Given by : The Odin Project**



###### **Description :** 

###### &#x20;                      The Project explores  This project explores the planet Earth and provides information about its structure, atmosphere, natural resources, and importance in supporting life. The purpose of the project is to present key facts about Earth in an engaging and easy-to-understand format while helping users learn about the planet we live on.



#### **Features**

* Introduction to Earth and its position in the Solar System
* Information about Earth's layers, including the crust, mantle, and core
* Details about the atmosphere and its composition
* Overview of oceans, continents, and natural resources
* Explanation of Earth's rotation and revolution
* Discussion of environmental challenges and conservation efforts
* Responsive and user-friendly design for easy navigation
* Technologies Used
* HTML for structuring content
* CSS for styling and layout
